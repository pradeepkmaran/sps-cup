Tracks & Problem Statements

üß†

Biomedical Signal Processing: Brain Computer Interface (BCI)

Motor Imagery EEG Control

Build a signal processing pipeline that classifies multi-channel EEG data based on imagined limb movements, such as left hand versus right hand. The solution should include preprocessing like filtering and artifact removal, feature extraction using CSP (Common Spatial Patterns) and wavelet transforms, and classification with methods like SVM (Support Vector Machine) and CNN (Convolutional Neural Network). Bonus points for real-time implementation or virtual interface integration.

Emotional State Recognition

Create a system that processes EEG data to classify emotional states such as happiness, stress, or calmness. Use datasets like DEAP and DREAMER, applying frequency band analysis, entropy measures, and machine learning classifiers. Focus on interpretability and subject-independent accuracy for mental health monitoring.

Cognitive Workload Estimation

Design a pipeline to analyze EEG recordings for measuring cognitive load during tasks like problem-solving or multitasking. Include preprocessing, feature engineering such as theta/beta ratio and spectral power, and use regression or classification models. Real-time feedback and adaptive systems are encouraged.

üì∂

Wireless Sensing: Wi-Fi Vision

Device-Free Activity Recognition

Use Channel State Information (CSI) from standard Wi-Fi packets to classify activities like walking, sitting, or waving without cameras or wearables. Analyze temporal and frequency features using signal processing and machine learning to ensure reliable, privacy-friendly activity recognition.

Non-invasive Sleep Stage Monitoring

Design a system using Wi-Fi CSI to distinguish sleep stages like light, deep, and REM sleep without wearables. Analyze subtle signal changes caused by respiration and movements using machine learning, offering real-time visualizations or sleep reports for actionable feedback.

Indoor Positioning & Navigation for the Visually Impaired

Use CSI and signal fingerprinting to estimate indoor locations, integrating accelerometer and gyroscope data for pedestrian dead reckoning. Provide obstacle detection via dynamic signal changes, delivering voice or haptic feedback for safe navigation while preserving privacy with minimal infrastructure.

Environmental & Geospatial Signal Processing

Seismic Event Classifier

Identify earthquakes by detecting clear P and S wave arrivals, long codas, and low-frequency waves, while distinguishing explosions and urban noise based on wave characteristics. The system should classify seismic events accurately using signal features.

Acoustic Source Localization

Set up a microphone array to record sound signals and calculate Time Difference of Arrival (TDOA) between microphones. Use triangulation techniques to determine the source location precisely, leveraging accurate environmental geometry.

Deforestation Detector with Radar Imagery

Use time-series SAR satellite images to detect vegetation loss by preprocessing, noise removal, and aligning datasets. Compare imagery over time to highlight deforestation areas, mapping changes in backscatter patterns for conservation efforts.

üîã

Electronic Vehicles

Inverter Fault Detection

Detect inverter faults by analyzing current and voltage waveforms using filtering, FFT, Wavelet Transform, and time-frequency analysis. Identify faults like harmonics and short circuits using adaptive filtering and machine learning for real-time alerts and fleet monitoring.

Electric Motor Condition Monitoring

Monitor EV motor health by analyzing vibration, current, and sound signals. Preprocess to enhance signal quality, apply Envelope Detection and Hilbert-Huang Transform, and use AI/ML models like SVM and CNN to detect faults, offering predictive maintenance insights.

Road Condition Classification

Classify road conditions using accelerometer signals from EV suspensions. Apply STFT and Wavelet Transforms to detect patterns linked to smooth, rough, or hazardous surfaces, enhancing ride comfort, safety, and fleet optimization.

Propose novel signal processing solutions addressing real-world challenges. Solutions may be hardware or software-based and must demonstrate technical depth, innovation, and practical impact. Interdisciplinary and cutting-edge approaches are encouraged.
